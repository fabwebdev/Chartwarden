# Chartwarden Quick Reference

## Ports
- Frontend: 3000
- Backend: 3001
- PostgreSQL: 5432
- Redis: 6379
- pgAdmin: 5050

## Key Directories
- Frontend: `apps/web/src/`
- Backend: `services/api/src/`
- Schemas: `services/api/src/db/schemas/`
- Routes: `services/api/src/routes/`
- Shared types: `packages/types/src/`

## User Roles
admin, physician, nurse, rn, lpn, cna, social_worker, chaplain, aide, billing, scheduler, medical_director

## Patient Statuses
pending_admission, active, discharged, deceased, revoked, transferred

## Levels of Care
routine, continuous, inpatient, respite
