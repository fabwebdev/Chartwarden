# Chartwarden Product Specification

## Vision
Build a comprehensive, HIPAA-compliant EHR system for hospice care providers.

## Core Features
- Patient Management
- Clinical Documentation (visits, HOPE, care plans, medications)
- Compliance (14-day IDG, certifications, F2F)
- Billing (claims, NOE, payments)
- Staff Management (credentials, caseload, scheduling)

## Non-Functional Requirements
- API response < 200ms (p95)
- Page load < 2 seconds
- 99.9% uptime
- HIPAA compliant
